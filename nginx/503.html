You are probably in one of these cases:

Cause Solution
1st case Too many people inside the same building as yours (or more precisely, at the same IP address) are currently accessing the website. Contact us, we will adjust that. If you know it, specify the amount of persons using the website at the same time at your address.
2nd case You are trying to do a denial-of-service attack. The website is protected against this type of attack. You see this error because you are about to get banned. Other users are not getting this error.
3rd case You are hitting the refresh button extremely fast. Don’t do that.
